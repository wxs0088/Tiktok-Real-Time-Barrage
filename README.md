# 抖音直播弹幕爬取 python 实现

## 环境

```
python 3.11.5
```

## 运行

```bash
pip install -r requirements.txt
python main.py
```

## 直播间id(room_id)获取

```
### xxxxxx 为直播间id
https://live.douyin.com/xxxxxx
```
